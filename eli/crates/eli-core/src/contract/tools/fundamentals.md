### `eli finance fundamentals`

Income statement, balance sheet, and cash flow essentials.

```bash
eli finance fundamentals --ticker AAPL
```

**Returns:**
- `total_revenue`, `gross_profit`, `operating_income`, `ebitda`
- `cash_and_equivalents`, `total_debt`
- `operating_cash_flow`, `free_cash_flow`

**Note:** Single ticker only (`--ticker` not `--tickers`).
