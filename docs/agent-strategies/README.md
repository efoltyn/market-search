# Agent Strategy Skills

This folder defines four execution strategies for `eli agent` workflows.

- `critique.md`: Attack and improve a lead thesis/report.
- `evidence.md`: Expand dataset depth and reduce uncertainty.
- `compete.md`: Independent answer race with scoring.
- `debate.md`: Multi-agent argument and consensus synthesis.

Use these as skill-style runbooks: copy the strategy prompt scaffold, run workers, and evaluate using the acceptance checks in each file.

Recommended fast/free model candidate to include in swarm rosters:
- `nvidia/nemotron-3-nano-30b-a3b:free`
